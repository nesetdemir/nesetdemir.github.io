HTML

html iskeletimiz diyebiliriz.
html dosyaların uzantıları .html olur. ilk dosyamızın adı sunucuların otomatik tanımasından dolayı genelde index.html olur.

html taglar <html> şeklinde açılır. </html> şeklinde kapanır.
html tagları açıldığı sırayla kapanırlar.
örneğin önce <html> sonra <head> tagı açıldıysa önce </head> sonra </html> kapatılır.
genel yapı aşağıdaki gibi olur.

<html>
		<head> </head>
		<body> </body>
  </html>
